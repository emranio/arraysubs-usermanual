# Manage Subscriptions

Use this section when you are working with subscriptions **after they already exist**.

This is the merchant/admin side of day-to-day subscription operations: finding subscriptions, creating them manually, editing key fields, reviewing billing history, and taking manual lifecycle actions.

## What belongs here

These pages explain how to:

- search the subscription list and export admin data
- create subscriptions manually from the admin area
- edit supported subscription fields after creation
- use the subscription detail screen for support, billing review, and lifecycle actions
- understand which subscription emails are controlled through WooCommerce email settings

## Start here

- [Subscription List, Search, and Export](./subscription-list-search-and-export.md) — find subscriptions quickly, understand the status filters, and use the available export options.
- [Create and Edit Subscriptions](./create-and-edit-subscriptions.md) — manually create subscriptions and learn what the current edit screen does and does not allow.
- [Subscription Detail Screen](./subscription-detail-screen.md) — review every major card, quick action, timeline, order table, and note area on the detail page.
- [Lifecycle Management and Manual Actions](./lifecycle-management-and-manual-actions.md) — understand status changes, scheduled cancellation, reactivation, and product-deletion continuity.
- [Subscription Emails and Notifications](./subscription-emails-and-notifications.md) — review the built-in customer/admin emails, template controls, and email-setting ownership.

## What does not belong here

This section does **not** cover:

- how to configure subscription products before purchase
- customer self-service actions inside My Account
- gateway setup and webhook behavior *(Pro)*
- checkout layout customization *(Pro)*
- retention, refund, plan-switching, or Store Credit deep dives

Those topics belong in their own owning sections so the merchant manual does not repeat the same settings in multiple places.

## Recommended reading order

1. Start with [Subscription List, Search, and Export](./subscription-list-search-and-export.md) if your team mainly handles support and operations.
2. Then read [Create and Edit Subscriptions](./create-and-edit-subscriptions.md) if staff manually set up subscriptions for customers.
3. Keep [Subscription Detail Screen](./subscription-detail-screen.md) open as the daily reference page.
4. Read [Lifecycle Management and Manual Actions](./lifecycle-management-and-manual-actions.md) before changing live statuses in production.
5. Use [Subscription Emails and Notifications](./subscription-emails-and-notifications.md) when configuring merchant/customer email behavior.
