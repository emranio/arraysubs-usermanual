# Create a Subscription Product
2. Short Overview

This guide explains how to create a subscription-based product in WooCommerce using the plugin. Use this when you want to sell recurring products or services.
3. Prerequisites (Optional but useful)

## Prerequisites
- WooCommerce installed and activated
- Subscription plugin installed
- At least one payment gateway configured
4. Step-by-Step Instructions (Core section)

This is the most important part. Keep steps clean and action-focused.

## Steps

1. Go to **Products → Add New**
2. Enter product title and description
3. Select **Subscription Product** from product type dropdown
4. Set billing interval (e.g., monthly, yearly)
5. Configure price and trial period
6. Publish the product
