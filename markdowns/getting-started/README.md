# Get Started

This section helps a merchant go from **installed plugin** to **first working subscription flow** without guessing which screen to open next.

Use these pages when you want to:

- confirm the store meets ArraySubs requirements
- activate the plugins in the correct order
- understand the billing and status model before configuring products
- create and test your first subscription product
- verify the admin experience and customer portal before going live

## Start here

- [Before You Launch](./before-you-launch.md) — requirements, activation order, core concepts, statuses, and the billing timeline.
- [Setting Up Your First Subscription](./setting-up-your-first-subscription.md) — configure General Settings, create the product, and prepare checkout.
- [Test Order and Portal Review](./test-order-and-portal-review.md) — place a test order, inspect the admin subscription, and review the customer portal.
- [Essential Daily Workflows](./essential-daily-workflows.md) — the subscriptions list, renewal lifecycle, common customer actions, and a go-live checklist.
- [Optional Pro Quick Start](./optional-pro-quick-start.md) — when to introduce Pro-only systems without overcomplicating initial launch.

## Recommended reading order

1. Read [Before You Launch](./before-you-launch.md) first.
2. Complete [Setting Up Your First Subscription](./setting-up-your-first-subscription.md).
3. Perform the checks in [Test Order and Portal Review](./test-order-and-portal-review.md).
4. Keep [Essential Daily Workflows](./essential-daily-workflows.md) open when training support or store operations staff.
5. Add [Optional Pro Quick Start](./optional-pro-quick-start.md) only after the core workflow already behaves correctly.

## Scope note

This section focuses on the **first successful subscription setup**.

Advanced topics such as refunds, plan switching, retention offers, Member Access rules, Checkout Builder *(Pro)*, Store Credit *(Pro)*, and Automatic Payments *(Pro)* are documented in their owning module sections instead of being repeated here.
