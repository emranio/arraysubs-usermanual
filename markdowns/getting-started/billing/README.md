# FAQ
### Is a signup fee the same as the recurring amount?
No. A signup fee is a separate one-time upfront charge.

### Why is billing inside getting started?
Because trial behavior, signup fees, and first-charge expectations are core onboarding topics for merchants setting up subscription products.
