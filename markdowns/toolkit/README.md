# User Guide
The **Toolkit** topic covers the free ArraySubs settings area used to clean up and control the member-facing access experience around WordPress admin screens.

You can reach this topic from:

- **ArraySubs → Settings → Toolkit**

Toolkit is meant for stores that want customers and members to stay in a polished frontend experience instead of drifting into raw WordPress backend pages.

In practical terms, this topic is about controlling things such as:

- whether regular users see the frontend WordPress admin bar
- whether non-admin users can access `/wp-admin`
- where blocked users should be redirected
- whether the default WordPress login page should stay visible
- which roles are still allowed to use the backend when restrictions are enabled

## How Toolkit fits into ArraySubs
Toolkit is a **free** settings topic in the core ArraySubs experience.

Unlike billing, renewals, and plan changes, Toolkit is mainly about store access flow and user experience. It helps merchants keep subscription customers inside the WooCommerce account experience instead of exposing standard WordPress admin screens unless that access is intentional.

This makes Toolkit especially useful for:

- membership-style stores
- online course sites
- client portals
- wholesale or members-only shops
- stores where customers should use **My Account** instead of `/wp-admin`

# Use Case
A membership merchant wants customers to log in through WooCommerce **My Account**, avoid seeing the frontend admin bar, and stay out of `/wp-admin` unless they have a trusted staff role. Toolkit gives the store one place to manage those access rules.

# FAQ
### Is Toolkit a free feature?
Yes. Toolkit is documented here as part of the free ArraySubs settings experience.

### Is Toolkit about subscription billing?
No. Toolkit is primarily about access control and frontend experience around WordPress admin areas.

### Who should read this topic?
Store owners, admins, and support teams who manage customer access rules and login flow.

### Does Toolkit replace WooCommerce My Account?
No. Toolkit works alongside WooCommerce My Account and is designed to push regular users toward that frontend account flow instead of backend WordPress screens.
